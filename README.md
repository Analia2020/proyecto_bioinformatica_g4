# proyecto_bioinformatica_g4
## Análisis de Expresión Génica con RNA-Seq

Repositorio para el análisis de expresión génica a partir de datos de secuenciación de ARN (RNA-Seq). El pipeline cubre desde el control de calidad de las lecturas crudas hasta el análisis de expresión diferencial.

**Pipeline**
Las muestras se procesan desde su forma cruda hasta identificar qué genes están activos y en qué cantidad en cada condición experimental.

**Requisitos**
Python 3.8+
R 4.0+
